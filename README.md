Machine Learning Projects Portfolio

This repository contains multiple machine learning projects showcasing predictive models using Python, Scikit-Learn, TensorFlow, and other ML tools. Each project is fully explained in its respective Jupyter Notebook with code, outputs, visualizations, and insights.

---

## 1. Iris Flower Prediction

**Overview:**  
A Logistic Regression model that predicts the species of an Iris flower based on sepal and petal measurements.

**Technologies Used:**  
Python, Pandas, NumPy, Scikit-Learn, Matplotlib


**Results:**  
- Accuracy: ~95–98%  
- Includes visualization and user input prediction function.

---

## 2. StockForecast — S&P 500 Movement Prediction

**Overview:**  
A Random Forest Classifier predicting whether the S&P 500 will go up or down the next day using historical data.

**Technologies Used:**  
Python, Pandas, NumPy, yfinance, Scikit-Learn, Matplotlib

**Results:**  
- Precision Score: ~54–57%  
- Includes backtesting and trend feature engineering.
